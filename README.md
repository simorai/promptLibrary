# promptLibrary

PromptLibrary is a Laravel + Inertia + Vue application for managing AI conversations, reusable prompts, threaded comments, and shareable exports in one workspace.

## What it does

- Create, pin, search, rename, and delete conversations.
- Store reusable prompts for quick insertion into chats.
- Add comments to messages and export conversations as markdown or PDF.
- Use a dashboard built from shared UI primitives for quick navigation and status checks.
